## 📋 運用ルール概要

### お金の管理
1. **フェーズ1（今）**: PayPay銀行 + PayPayカード + freee
2. **フェーズ3（法人化後）**: 住信SBIネット銀行を追加し、給与等の支払いを行う
3. 自動連携を活用し、手入力は最小限に

### GitHub運用
1. **経営判断**: 会社GitHubに直接コミット
2. **実験・仮説**: 個人GitHubで自由に試行錯誤
3. **プロジェクト**: プルリクエスト運用
